# Dodge-The-Blocks
I made a game where you dodge blocks, and for each second you survive, you get a point. Every 10 seconds a power up spawns.
If you want to try it out, run main.py.

# Controls
Click any key to start when you first open the game. Use the arrow keys to move left and right. Click enter to try again when you lose.

# Power Ups
This game also has 5 power ups. You activate a power up by touching it. The five power ups are as follows: Speed upgrade, which increases your speed, Speed downgrade, which decreases your speed, Size upgrade, which increases your size, Size downgrade, which decrases your size, and Point upgrade, which gives you 5 points. Speed upgrade is a light blue, Speed downgrade is a dark blue, Size upgrade is a light green, Size downgrade is a dark green, and Point upgrade is a goldish yellow.

# Installation
To install, you need python 3.11 installed and on PATH. If you have that, do the following command:
python3.11 -m pip install pygame

# Playing the game
After installing python 3.11 and using pip to install pygame, complete the following command, or run main.py using python3.11:
python3.11 main.py
