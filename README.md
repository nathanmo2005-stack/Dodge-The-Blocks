# Dodge-The-Blocks
I made a game where you dodge blocks, and for each second you survive, you get a point. Every 10 seconds a power up spawns.
If you want to try it out, run main.py.

# Controls
Click any key to start when you first open the game. Use the arrow keys to move left and right. Click enter to try again when you lose.

# Power Ups
This game also has 8 power ups. You activate a power up by touching it. The eight power ups are as follows: Speed upgrade, which increases your speed, Speed downgrade, which decreases your speed, Size upgrade, which increases your size, Size downgrade, which decrases your size, Point upgrade, which gives you 5 points, teleport left, which gives you the ability teleport left, teleport right, which gives you the ability teleport right, and dash, which gives you the ability dash.. Speed upgrade is a light blue, Speed downgrade is a dark blue, Size upgrade is a light green, Size downgrade is a dark green, Point upgrade is a goldish yellow, teleport left has a arrow to the left, teleport right as an arrow to the right, and dash has an orange dashing box.

# Abilities
Right now, this game as 3 abilities. Teleport left, teleport right, and dash. 
- Teleport Left: When you click SPACE, you teleport to the left.
- Teleport Right: When you click SPACE, you teleport to the right.
- Dash: When you click SPACE, depending on which direction you last moved to, you dash in that same direction.

# Installation
To install, you need python 3.11 installed and on PATH. If you have that, do the following command:
python3.11 -m pip install pygame

# Playing the game
After installing python 3.11 and using pip to install pygame, complete the following command, or run main.py using python3.11:
python3.11 main.py
